astate.util.re_find_in_body(body, cond: ~typing.Callable | type = <class 'ast.FunctionDef'>, max_levels: int | None = None)[source]

Recursively find elements in an ast body that satisfy a condition

Parameters:

  • body – A list of ast nodes or object that can be resolved to it. See ensure_body to see what objects can be resolved to a body.

  • cond – The condition to apply to ast elements of body to determine whether to yield or not

  • max_levels – Maximum levels of recursion desired

Returns:

A generator of filtered ast nodes

>>> def foo(x):
...     def bar(y):
...         y * 1
...     return bar(x)
>>>
>>> [x.name for x in re_find_in_body(foo)]
['foo', 'bar']
>>> [x.name for x in re_find_in_body(foo, max_levels=1)]
['foo']
